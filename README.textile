h2. Keys

* A: @W@ (or @Z@ on QWERTY keyboards)
* B: @X@
* Start: @Enter@
* Select: @Backspace@
* Directional keys: Arrows

h2. How to compile

* Install @CMake@, @SDL2@, @SDL2_image@, @SDL2_mixer@, @glm@, @OpenGL@ (=> 2.1) and @glew@ (only required for Windows)
* *GCC Users:* There's a "bug":https://gcc.gnu.org/bugzilla/show_bug.cgi?id=55914 in @GCC@ < 4.9, so the code won't compile. Please install @GCC 4.9@.
* Run @mkdir build && cd build && cmake ..@ at the root of the game folder
* Run @make@, wait, and enjoy!
